# Kurulum

Projeyi klonlayın:
```
git clone https://github.com/kullaniciadi/event-planning-frontend.git
cd event-planning-frontend
```
Gerekli bağımlılıkları yükleyin:
```
npm install
```
React uygulamasını çalıştırın:
```
npm start
```
## Özellikler

Kullanıcı girişi ve kaydolma

Etkinlikleri listeleme ve detay sayfası

Kullanıcının ilgisine göre önerilen etkinlikleri gösterme

Etkinliklere katılma ve mesaj gönderme

Rota hesaplama ve harita desteği

## Teknolojiler

React.js (Frontend framework)

React Router (Sayfa yönlendirmeleri)

Axios (API istekleri)

Leaflet.js (Harita entegrasyonu)

Bootstrap / Tailwind CSS (Stil düzenlemeleri)
